Method Note

No GitHub REST API

The dataset is collected from public GitHub HTML pages, issue pages, Atom feeds, contributor graph resources loaded by GitHub web pages, GitHub Security Advisory search pages, NVD public search pages, and OpenSSF Scorecard pages. Scores are operational proxies for comparing public maintenance signals; they are not direct code-quality or security-audit labels.
